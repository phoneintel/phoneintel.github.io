Introduction

Welcome to the documentation for PhoneIntel. This tool processes phone numbers to perform various actions, including fetching information, searching, and generating dorks.

PhoneIntel is a powerful tool for cybersecurity researchers, OSINT investigators, and anyone needing detailed information about phone numbers. It provides a simple and efficient interface for gathering and analyzing data.

Installation

To install the project, use the following command:

pip install phoneintel

After installation, you can use the phoneintel command to interact with the tool.

Usage

General Command

phoneintel <option> <phonenumber>

--search

This parameter should be combined with:

  • --string: Pass a string with phone numbers or text containing phone numbers to extract and analyze.
  • --input: Upload a txt file with a hidden number or a list of numbers to find and analyze.

--dorks

This parameter must be accompanied by --type followed by one of the following options:

"social_networks", "forums", "classifieds", "ecommerce",
"news", "blogs", "job_sites", "pastes", "reputation", 
"phone_directories", "people_search", "all"

This will provide information about the phone number and create specialized dorks for advanced Google searches.

--map

Any of the main commands (--info, --search, --dorks) can be supplemented with the --map parameter, which will open a default browser window using OpenStreetMap to show the country and area location of the phone line.

--neutrino

phoneintel --neutrino [ | --login --id  --key  | --show] [--map]

This option allows interaction with the Neutrino API:

  • <phone>: Phone number to be processed
  • --login: Login to the Neutrino API
  • --id: API ID
  • --key: API KEY
  • --show: Show Neutrino API Credentials
  • --map: Execute map function (optional)
